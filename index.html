What NetMon stores locally

SQLite netmon.db (always on) — packets metadata (ts, src, sport, dst, dport, proto, length, pid, process, domain), DNS answers, blocked IPs, alerts, rules.
In-memory — active flows (pruned 5 min idle), process counters, per-IP GeoIP cache (≤50k), bandwidth buckets.
PCAP = packet capture file (Wireshark format, .pcap) containing every byte on the wire. Off by default; click to start. PCAPs are big — only enable if you need byte-level inspection.
No data leaves your machine except GeoIP lookups to ip-api.com (disable in Settings).
